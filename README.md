# 🍽️ Restaurant Digital Suite

## 🇬🇧 English

### Overview

A modular digital system designed for restaurants to manage menus, allergens, translations, and reservations in a simple and automated way.

### Problem

Restaurants often struggle with:

* managing allergen information correctly
* offering multilingual menus
* handling reservations efficiently
* reducing manual work

### Solution

This project is a suite of custom WordPress plugins that provides:

### 🔹 Allergen Filtering System

* Full allergen mapping per dish
* Dynamic filtering for users
* Clear and accessible interface

### 🔹 Multilingual Menu Translation

* JSON-based translation system
* Language mapping via structured arrays
* Scalable multi-language support

### 🔹 Automated Dish Management

* Standardized data structure
* Simplified content management

### 🔹 WhatsApp Booking Integration

* Direct reservation flow
* Fast and frictionless UX
* No external platforms required

### Tech Stack

* WordPress (custom plugins)
* PHP
* JavaScript
* JSON data mapping

### Architecture

* Modular plugin-based system
* Data-driven structure using JSON
* Dynamic frontend interaction
* Optimized backend for restaurant usage

### Demo

👉 https://www.siti4ristoranti.it/

### Results

* Improved user experience
* Reduced allergen-related errors
* Better accessibility for international customers
* Increased booking conversion
